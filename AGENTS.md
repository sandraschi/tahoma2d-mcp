# tahoma2d-mcp — Session Log

## Server Info
- Backend: 11013 (FastMCP HTTP /mcp)
- Frontend: 11012 (Vite)
- Stack: FastMCP 3.2, React 18, Tahoma2D 1.6+

## Setup
```bash
uv sync
# Tahoma2D must be installed at tahoma2d.org
```

## Commands
- `just lint` — ruff check
- `just fix` — ruff fix + format
- `just serve` — run MCP server
- `just web` — start frontend
- `just start` — full stack
- `just test` — pytest

Install docs: follow mcp-central-docs/standards/AGENT_INSTALL_REFERENCE.md
