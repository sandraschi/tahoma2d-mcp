# Tauri Native Desktop

This repo includes a Tauri 2.0 wrapper that bundles:
- The React webapp (built)
- The Python backend (compiled via PyInstaller)

Build: `just build-all`

The installer lands at `native/target/release/bundle/nsis/`.

See [tauri_godot_sota.md](../../../mcp-central-docs/standards/rules/tauri_godot_sota.md) for the fleet standard.
